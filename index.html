Here are some of my favorite projects

Voice controlled Tello Drone

A web application hosted on a NodeJs server that communicates with a Tello drone via UDP. The application frontend is a simple website that parses voice commands then packages them into Json messages to send to the backend via a websocket.
GithubGithub

JavaBot - sensor library

A Java sensor library for the Raspberry Pi, a single board computer. This library is support the PICAM, GPS NMEA parsing through UART, lidar Lite v3 from Garmin through I2C, servo and esc control through PWM and HRUSB-Max Sonar-EZ ultrasonic rangefinders through USB.
GithubGithub

Weather Channel for Mars

A NodeJs web application that display weather data for Mars using a template view engine called Pug. The web app query NASA's InSight: Mars Weather Service API.
